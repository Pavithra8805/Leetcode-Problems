# Leetcode-Problems

Daily LeetCode practice repository.

## 🎯 Goal

Solve LeetCode problems **every day** to improve:

- problem-solving speed
- data structures & algorithms fundamentals
- interview readiness
- coding consistency

## 📅 Daily Challenge Commitment

- Solve at least **1 problem per day**
- Write clean, readable solutions
- Add time/space complexity notes
- Review and improve older solutions weekly

## 🧠 Solution Format (Suggested)

For each problem, include:

1. Problem name and link
2. Approach summary
3. Time complexity
4. Space complexity
5. Final code

Example:

```md
## 1. Two Sum
Link: https://leetcode.com/problems/two-sum/

Approach:
- Use a hash map to store visited numbers and indices.

Time Complexity: O(n)
Space Complexity: O(n)
```
